# JobSearch aka Jobhunter(versão pública)

Este repositório contém apenas os ficheiros públicos do meu projeto JobSearch, disponível em:
https://jobsearch.alvarofr.pt/

Inclui apenas:
- Templates HTML (Jinja)
- Ficheiros estáticos (CSS/JS/imagens)

O backend, scrapers, base de dados e lógica interna foram removidos por questões de segurança e privacidade.


<img width="955" height="743" alt="image" src="https://github.com/user-attachments/assets/34be25a4-a608-4d05-9e7a-870283d68fce" />

JobSearch é um agregador de ofertas de emprego desenvolvido por mim, reunindo vagas de várias fontes num único lugar. Esta versão pública inclui apenas os templates e ficheiros visuais do site — toda a lógica interna, scrapers e base de dados permanecem privados por motivos de segurança.
